# What is VAHEAT
The VAHEAT is a temperature controller from Interherence that is designed to provide precise temperature control for a variety of applications. It features a digital display, adjustable temperature setpoints, and a variety of control modes including on/off, proportional, and PID. It also has a built-in alarm system and can be used with a variety of sensors. The VAHEAT is designed to be easy to install and use, and is suitable for a wide range of applications.

The VAHEAT temperature controller from Interherence has an API available via serial communication. This API allows users to control the temperature of the device remotely, as well as access data from the device such as temperature readings, alarms, and other settings. The API also allows users to configure the device to their specific needs, such as setting temperature thresholds, enabling/disabling alarms, and more.
